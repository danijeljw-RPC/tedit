# MVP-003 — File UI and Syntax Foundation

## Objective

Add discoverability and first syntax support.

## Scope

- menu bar model
- File menu
- Edit menu
- Search menu
- View menu
- Tools menu
- Help menu
- open file prompt
- save as prompt
- basic file browser
- line numbers
- whitespace display toggle
- tab insertion setting: literal tab, 2 spaces, or 4 spaces
- native syntax token model
- simple built-in highlighters
- nano syntax importer tier 1

## Acceptance criteria

- menus can be opened with Alt shortcuts
- open/save-as works without command-line path
- native highlighting model is renderer-independent
- nano syntax parser can load simple syntax rules
- all MVP-003 behavior configures, builds, and tests on macOS, Linux, and Windows
