# Search, Replace, Highlight

## Find

Required:

- find prompt
- find next
- find previous
- highlight all visible matches
- status message showing match count if available

Options later:

- case-sensitive toggle
- whole-word toggle
- regex mode
- incremental search

## Replace

Required later:

- replace current
- replace all
- confirmation before replace all
- undo grouping for replace all

## Highlighting

Search highlighting is separate from syntax highlighting. Search highlights should visually override syntax colours.
